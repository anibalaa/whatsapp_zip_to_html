# whatsapp_zip_to_html
Pequeño script hecho con Claude para convertir un chat exportado de WhatsApp a un página Web con todos los archivos multimedia

https://chat-zip-viewer.netlify.app/

Este proyecto es un conversor de chats de WhatsApp a HTML, diseñado para usuarios sin conocimientos técnicos que solo disponen de móvil o tablet.
Qué hace:

Permite subir el archivo ZIP que genera WhatsApp al exportar un chat, y lo convierte en una página HTML navegable con las imágenes, audios y vídeos embebidos. Todo el procesado ocurre en el navegador del usuario, sin servidores, sin subir datos a ningún sitio.
A quién va dirigido:

Usuarios no técnicos, principalmente desde móvil Android o tablet. El lenguaje debe ser siempre claro, sin tecnicismos, con instrucciones paso a paso.
Cómo se usa:

Abrir WhatsApp → entrar al chat → tres puntos → Más → Exportar chat → Con archivos multimedia
Guardar el ZIP resultante en el almacenamiento del dispositivo
Acceder a esta web desde el navegador del móvil o tablet
Pulsar el botón de subir archivo y seleccionar el ZIP exportado
Esperar a que procese y descargar el HTML generado
Abrir ese HTML con el navegador para ver el chat completo con todos sus archivos multimedia

Restricciones importantes:

No almacenamos ningún dato. Todo es local en el dispositivo del usuario.
El HTML generado puede pesar bastante si el chat tiene muchas fotos o vídeos. Es normal.
Si el archivo ZIP es muy grande (más de 500MB), puede tardar varios minutos en procesarse desde móvil.
Compatible con Chrome y Firefox en Android. En iOS puede haber limitaciones de memoria con chats muy pesados.

Tono y estilo:

Cercano, directo, sin rodeos. Si el usuario tiene un problema, dar soluciones concretas, no explicaciones genéricas.
